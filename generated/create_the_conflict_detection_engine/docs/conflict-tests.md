# Conflict Detection Engine — Test Suite

## Overview

This directory contains the test suite for the Job-Star conflict detection engine.
The tests cover all four conflict types and the full detection pipeline.

## Test Files

| File | Description |
|------|-------------|
| `test_duplicate.py` | Unit tests for duplicate detection |
| `test_contradiction.py` | Unit tests for contradiction detection |
| `test_resource.py` | Unit tests for competing resource detection |
| `test_tension.py` | Unit tests for tension detection |
| `test_cross_domain.py` | Unit tests for cross-domain awareness |
| `test_storage_integration.py` | Tests for storage layer integration |
| `test_conflict_reporting.py` | Tests for scoring and reporting |
| `test_integration.py` | Full pipeline integration tests |
